Books (Work In Progress)

At martinade, we are heavy readers and the reading list is increasing. It's becoming difficult to follow who is reading/owning what. We are looking for a fun solution to this problem.

The following features are needed

  • Users should be able to create a reading list (toread, reading, done).
  • A book reference can be "own" by one or more users.
  • Book references can have "tags" (like sf, romance, politology, philo, ...)
  • Book references can be fetched from bnf api or other websites
